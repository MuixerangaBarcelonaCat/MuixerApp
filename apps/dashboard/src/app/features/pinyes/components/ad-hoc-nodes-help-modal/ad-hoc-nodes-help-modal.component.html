Nodes ad-hoc

Els nodes ad-hoc permeten afegir posicions extra directament al canvas d'assignacions, sense modificar el template original de la figura.

Com funciona?

  • Fes clic al botó + a la part inferior dreta.
  • Selecciona la categoria (Pinya o Decoratiu) i el tipus de posició.
  • Fes clic al canvas per col·locar el node.
  • Per als nodes de pinya, assigna una persona com qualsevol altre node.

Nodes de Pinya

Afegeixen posicions assignables (mans, agulla, cordó obert, etc.) o un "Comodí" amb nom lliure. Es comporten com qualsevol altre node del template: pots assignar-hi persones.

Nodes Decoratius

Serveixen per afegir elements visuals d'orientació o referència (rectangles, fletxes, cercles). No es poden assignar a persones. Són útils per marcar zones de l'escenari, direccions, o punts de referència.

Gestió dels nodes ad-hoc

  • Moure: arrossega el node al canvas.
  • Redimensionar/rotar: doble clic per activar els controls.
  • Propietats: selecciona el node per veure el panell de propietats (etiqueta, forma, color...).
  • Eliminar: selecciona el node i prem Supr o Retrocés.

Aspectes importants

  • Els nodes ad-hoc es mostren amb vora discontínua per distingir-los.
  • Els nodes decoratius es mostren amb fons transparent i menor opacitat.
  • Reinicialitzar la figura eliminarà també tots els nodes ad-hoc.
  • En importar una pinya, els nodes ad-hoc de l'origen es clonen automàticament.
  • Editar el template no afecta els nodes ad-hoc creats (i viceversa).
