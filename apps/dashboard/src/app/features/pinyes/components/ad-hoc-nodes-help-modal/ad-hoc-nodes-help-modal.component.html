Nodes ad-hoc

Els nodes ad-hoc permeten afegir posicions extra directament al canvas d'assignacions, sense modificar el template original de la figura.

Com funciona?

  • Fes clic al botó + a la part inferior dreta.
  • Selecciona el tipus de posició (mans, agulla, cordó obert, etc.) o "Comodí" per escriure un nom lliure.
  • Fes clic al canvas per col·locar el node.
  • Assigna una persona com qualsevol altre node.

Gestió dels nodes ad-hoc

  • Moure: arrossega el node al canvas.
  • Redimensionar/rotar: doble clic per activar els controls.
  • Eliminar: selecciona el node i prem Supr o Retrocés.

Aspectes importants

  • Els nodes ad-hoc es mostren amb vora discontínua per distingir-los.
  • Reinicialitzar la figura eliminarà també tots els nodes ad-hoc.
  • En importar una pinya, els nodes ad-hoc de l'origen es clonen automàticament.
  • Editar el template no afecta els nodes ad-hoc creats (i viceversa).
