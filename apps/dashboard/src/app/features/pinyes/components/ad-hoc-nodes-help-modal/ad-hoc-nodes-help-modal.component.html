Nodes ad-hoc

Els nodes ad-hoc permeten afegir posicions extra directament al canvas d'assignacions, sense modificar el template original de la figura. Es distingeixen per la seva vora discontínua (guions).

Tipus de nodes

  • Pinya — Posicions assignables (agulla, mans, laterals, vents, cordó obert, tap, crossa, contrafort) o un Comodí amb nom lliure.
  • Decoratiu — Elements visuals d'orientació (rectangles, fletxes, cercles). No assignables.
  • Direcció — Indiquen l'orientació de la figura o xicalla. Assignables (la persona indica cap on s'orienta).

Dreceres de teclat

Interacció bàsica
  • Seleccionar node (per assignar) Clic
  • Moure node ad-hoc Arrossegar
  • Obrir propietats del node Doble clic
  • Avançar al següent node buit Tab
  • Cancel·lar acció / sortir de mode Escape
Edició de nodes ad-hoc
  • Copiar node ⌘/Ctrl + C
  • Enganxar node ⌘/Ctrl + V
  • Duplicar node ⌘/Ctrl + D
  • Eliminar node Supr / Retrocés
  • Moure node (1 px) ↑ ↓ ← →
  • Moure node (10 px) Shift + ↑ ↓ ← →
Creació ràpida
  • Crear node de pinya (preset ràpid) ⌘/Ctrl + 1…8
  • Crear node Comodí (demana etiqueta) ⌘/Ctrl + 9

Coses importants

  • Reset esborra TOT: reinicialitzar la figura elimina tots els nodes (template + ad-hoc) i totes les assignacions.
  • El bloqueig s'aplica: quan l'event està bloquejat, no es poden afegir, moure ni eliminar nodes ad-hoc.
  • L'import copia els ad-hoc: quan importes assignacions d'una altra instància, els nodes ad-hoc es clonen automàticament amb les assignacions.
  • Canvis al template no afecten: si es modifica el template base, cap node d'aquesta instància canvia.

Comodí

El Comodí permet crear una posició amb un nom personalitzat per a situacions no estàndard (p.ex. "reforç esquerre", "lateral-agulla"). En seleccionar-lo, l'aplicació demanarà un nom abans de col·locar-lo al canvas.
